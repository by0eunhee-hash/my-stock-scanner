# Stock Flow AI - 한국투자증권 API 기반 주식 조건검색 대시보드

한국투자증권 OpenAPI를 활용해 조건검색 결과를 조회하고, 종목별 등락률/거래량/거래대금 정보를 대시보드 형태로 제공하는 개인 프로젝트입니다.  
조회된 후보 종목을 기반으로 Gemini API에 분석 요청을 보내 AI 리포트를 생성하는 기능을 구현했습니다.

> 본 프로젝트는 개발 학습 및 포트폴리오 목적의 프로젝트이며, 실제 투자 판단을 목적으로 하지 않습니다.

## 주요 기능

- 한국투자증권 OpenAPI 인증 및 조건검색 목록 조회
- 조건별 종목 리스트 조회
- 종목별 현재가, 등락률, 거래량, 거래대금 표시
- 카테고리별 필터링 및 조건 탭 전환
- 수급 에너지 지표 시각화
- Gemini API 기반 AI 분석 리포트 생성
- API 사용량 초과 및 오류 상황에 대한 사용자 안내 처리

## 기술 스택

- Frontend: React, Vite, JavaScript, CSS
- Backend: Node.js, Express
- API: 한국투자증권 OpenAPI, Gemini API
- Library: Axios, dotenv, cors

## 개발 과정에서 해결한 문제

한국투자증권 API 연동 과정에서 기존 예제와 실제 API 요청 방식이 달라 정상적으로 데이터가 조회되지 않는 문제가 있었습니다. 공식 문서를 확인해 인증 요청과 조건검색 API 호출 방식을 수정했고, 환경변수를 통해 API Key와 Secret을 분리하여 관리했습니다.

또한 AI 도구가 제안한 코드가 최신 API 방식과 맞지 않는 경우가 있어, 응답 구조와 오류 로그를 직접 확인하며 연동 로직을 보완했습니다. 이를 통해 AI 도구를 활용하되 결과를 검증하고 수정하는 개발 방식의 중요성을 경험했습니다.
